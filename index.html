TIFF2DJVU

This software is used to convert and merge multiple TIFF files to a single DJVU file.
Operating system : Ubuntu
Prerequisite packages : libtiff-tools, pdftk and pdf2djvu

Usage Instructions

Download any one of the compressed files, extract it and run the python file using "python tiff2djvu.py" in the terminal.
